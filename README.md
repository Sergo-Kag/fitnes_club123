🎯 О ПРОЕКТЕ
Фитнес-Клуб — это полноценное мобильное приложение для автоматизации работы фитнес-клуба. Приложение поддерживает три роли пользователей (администратор, тренер, клиент) и предоставляет широкий функционал для управления тренировками, абонементами, платежами и отзывами.

Ключевые особенности:
✅ Ролевая модель с разграничением прав доступа

✅ Система абонементов с возможностью продления, апгрейда и заморозки

✅ Бронирование тренировок в реальном времени

✅ Отзывы и рейтинги тренеров и тренировок

✅ Платежная система (виртуальные платежи)

✅ Отчеты и аналитика для администратора

✅ Уведомления-напоминания о тренировках

✅ Профили пользователей с загрузкой фото

Пошаговая установка
Клонируйте репозиторий

bash
git clone https://github.com/your-username/fitness-club.git
Откройте проект в Android Studio

File → Open → выберите папку с проектом

Синхронизируйте зависимости

Нажмите "Sync Now" в появившемся уведомлении

Или выберите File → Sync Project with Gradle Files

Соберите проект

Build → Make Project (Ctrl+F9)

Запустите приложение

Выберите устройство/эмулятор

Нажмите Run (Shift+F10)

📖 РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Подробное руководство пользователя доступно в файле USER_GUIDE.md или в приложении через пункт меню "Помощь" (скоро).

Краткая инструкция:

Для клиента:
Зарегистрируйтесь как клиент

Купите абонемент в разделе "Абонементы"

Записывайтесь на тренировки в разделе "Доступные тренировки"

Отслеживайте записи в "Мои записи"

Оставляйте отзывы после тренировок

Для тренера:
Зарегистрируйтесь как тренер (или получите данные от администратора)

Создавайте тренировки в разделе "Создать тренировку"

Смотрите расписание в "Мое расписание"

Отслеживайте клиентов в "Мои клиенты"

Для администратора:
Войдите под учетной записью admin

Управляйте тренерами и абонементами

Смотрите отчеты и статистику

Отслеживайте все платежи
